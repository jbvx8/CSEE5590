ICP 1: How to create a git repository

There are several steps involved in creating a repository:

  • Install github desktop app
    1. If not using the command line or IDE tools, the github GUI is useful for version control
    2. Download here
    3. Open desktop app

  • Create repository
    1. Click on the "+" icon
    2. Click the "Create" tab
    3. Enter name and path for local repository

  • Push local repository to remote
    1. Click "Publish" to push the locally created repository onto the github site
    2. You must be logged in to github for this to work
    3. The new repository should be visible on the remote github website
